# FUNERAL Frontend

Vue 3 + Element Plus frontend for FUNERAL OCI Registry

## Project Setup

```bash
cd funeral-frontend

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

## Development

The frontend will run on http://localhost:3000 and proxy API requests to the backend (http://localhost:8911).

## Features

- Browse and manage OCI repositories
- View repository tags and manifests
- Copy pull commands for images
- Upload guidance for pushing images to the registry
